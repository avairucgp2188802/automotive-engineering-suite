### How to Create a New Vehicle Model

1. Launch Automotive Engineering Suite
2. Click **"New Project"**
3. Choose template (Sedan, SUV, Sports Car)
4. Open **"CAD"** module
5. Sketch main body outlines
6. Extrude and add features
7. Save as `.vehicle` file

### How to Run Vehicle Dynamics Simulation

1. Open **"Vehicle Dynamics"** module
2. Load your CAD model or use example
3. Set vehicle parameters (mass, CG, wheelbase)
4. Choose test: Handling, Braking, Acceleration
5. Click **"Run Simulation"**
6. View results: Graphs and 3D animation
7. Export data to CSV

### How to Perform Aerodynamics (CFD) Analysis

1. Open **"Aerodynamics"** module
2. Import CAD body model
3. Set simulation domain (airflow box)
4. Choose velocity (e.g., 30 m/s = 108 km/h)
5. Click **"Generate Mesh"** (wait 2-5 min)
6. Click **"Run CFD"** (wait 5-15 min)
7. View results: Cd, downforce, pressure maps

### How to Design Suspension

1. Open **"Suspension Design"** module
2. Choose suspension type (McPherson, Double Wishbone)
3. Enter hardpoints (3D coordinates)
4. Adjust camber, caster, toe curves
5. View kinematic animation
6. Export geometry to CAD format

### How to Use CAN Bus Tools

1. Connect USB-to-CAN adapter (e.g., PeakCAN, Kvaser)
2. Open **"CAN Tools"** module
3. Select bitrate (500 kbps typical)
4. Click **"Start Monitoring"**
5. CAN messages appear in real-time
6. Create DBC files to decode signals
7. Send custom messages for ECU testing
